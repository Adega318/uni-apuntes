# HTTP
## Petición
Formato:
- Línea de petición, método + URL + versión
- Líneas de cabecera
- Línea en blanco
- Cuerpo de entidad
## Respuesta
Formato:
- Línea de estado, versión + [código de estado](https://en.wikipedia.org/wiki/List_of_HTTP_status_codes)
- Líneas de cabecera
- Línea en blanco
- Cuerpo de entidad
