# Requisitos
- HTTP 1.0
- Multihilo
- Metodos GET y HEAD

# Peticiones
## GET
Formato de petición:
1. Línea comenzada por un GET seguido del fichero.
2. N líneas con cabeceras.
3. Línea finalización en blanco.
Formato de respuesta:
1. línea de respuesta.
2. N cabeceras.
3. línea en blanco.
4. Contenido.
