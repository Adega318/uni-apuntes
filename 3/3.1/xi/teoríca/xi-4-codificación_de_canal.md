# Códigos
Un código binario de longitud n y cardinalidad $2^k$ es una colección de $2^{k}$ elementos.
$C(k,n)=\{c^{1},c^{2},...,c^{2^{k}}\},c^{m}$
Los elementos $c^{m}$ son las palabras código, con n como longitud de bloque.
