## Definición
Un proyecto se trata de un conjunto de :
- Recursos
- Organización
- Productos
- Objetivos
- Entorno de riesgo

Si estos elementos son cotidianos o operaciones estándar de la organización, a estos estándar se les considera operaciones normales y no son proyectos.
